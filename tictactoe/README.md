# Tic tac toe game

Tic tac toe game to be played with X and O

 ## Why use it? 

To play the game

 ## Features

 * Once a move was made it stays locked in
 * After game end it shows who won X, O or if it was a draw

 ## Built with

 * html
 * css
 * javascript
